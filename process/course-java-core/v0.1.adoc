== Beginner lessons (First lesson)

WARNING: Первое задание не является образцом заданий, которые будут выполняться в дальнейшем. Действия или шаги, которые будут выполняться в нем, необходимы для более глубокого понимания процесса работы с *source code* (*исходный код*) на java. По мере освоение нового материала, будут использоваться более продвинутые техники и инструменты.

* Установить link:https://youtu.be/ilhpwkcvEo0[OpenJDK 11^]
* Сделать *fork* репозитория link:https://github.com/rakovets/course-java-core[`course-java-core`^]
* Выполнить задание 'Intro' из этого репозитория, для этого нужно:
** Написать код.
** Скомпилировать.
** Запустить программу.
** Убедиться в ее корректной работе.
** Загрузить исходный код программы в репозиторий, который Вы получили после *fork*, в ветку `intro` с помощью *GUI* web-сервиса *GitHub*.

NOTE: Для корректного выполнения задания следует посмотреть видео link:ttps://youtu.be/Li40OBvq5Tg[How to do first homework?^].
