= Terminologies

== Abbreviations

* *GUI* - *Graphical User Interface*/*Графический интерфейс пользователя*.
* *IDE* - *Integrated Development Environment*/*Интегрированная среда разработки*
* *MR* - *Merge Request*/*Запрос на слияние*.
* *PR* - *Pull Request*/*Запрос на слияние*.
* *VCS* - *Version Control System*/*Система Контроля Версий*.

== Definitions

* *branch*/*ветка* - параллельная, независимая копия кода в данном репозитории.
* *commit* - фиксирование изменений в *local repository*.
* *fork* - сделать копию данного *repository* в свои *repositories* на git-hosting.
* *git-hosting* - внешний специальный сервис для хранения и работы с *git-repositories*.
* *local repository*/*локальный репозиторий* - отдельная директория для хранения кода и его истории, где для контроля версий используется *git*.
* *remote repository*/*удаленный репозиторий* - репозиторий который находится у некоторого провайдера для размещения исходного кода (например: *GitLab*, *GitHub*, *Bitbucket*).
